## 목표 및 기능
- 사용자들은 친환경 실천 행동을 기록하고 공유할 수 있습니다.
- 친환경 실천을 통해 사용자들은 서로를 독려하고 함께 환경 보호에 기여할 수 있습니다.
- 자신의 친환경 실천 내역을 확인하고 발전시킬 수 있는 개인 로그를 관리할 수 있습니다.
